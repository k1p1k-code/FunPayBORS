# Использование плагинов
*Лоуд плагинов с python скрпита экспереминтальная*
___
1. Установите python от 1.13.2 до 1.13.6
2. Создайте папку "plugins" рядом с exe скрипта
4. Создайте [виртуальное окружение](https://pythonchik.ru/okruzhenie-i-pakety/virtualnoe-okruzhenie-python-venv) с названием venv важно 
5. Скачайте или создайте plugin который хотите использовать
5. Распакуйте папку где plugin.py в папку "plugins"
6. Скрипт сам подхватит все
